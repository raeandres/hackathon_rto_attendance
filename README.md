# hackathon_rto_attendance
A small hackathon project for RTO attendance


## Pre-req
### Install Node Js on your machine
https://nodejs.org/en/ -> Select LTS for stable release

### Install MongoDb on your machine
#### on Mac (via Homebrew):
https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/

#### on Windows:
https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/

## requirements
npm -i after cloning to install necessary modules.



## To Start Mongodb:
### on Mac
https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/#run-mongodb-community-edition

### on Windows
https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-windows/#run-mongodb-community-edition-from-the-command-interpreter



## To start NodeJS app:
npm run dev


go to http://localhost:4000
